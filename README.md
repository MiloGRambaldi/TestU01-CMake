TestU01-CMake
=============

TestU01 Library including a CMake build system.

This is a CMake build system for the <a href="http://www.iro.umontreal.ca/~simardr/testu01/tu01.html">TestU01 library</a> 
for the empirical statistical testing of uniform random number generators. The TestU01 library 
is the work of Pierre L'Ecuyer and Richard Simard.

Included with the build system is TestU01-1.2.3, version 1.2.3 of the TestU01 
library that was created on 18 August 2009. 

Please see [README](README) for more information on TestU01.
